دليل الـ API المباشر

استخدم الروابط التالية لإرسال رسائل من خلال تطبيقك الخارجي.

POST /api/sendText

إرسال رسالة نصية عبر رقم مسجل.

{
  "senderPhone": "YourConnectedNumber",
  "number": "TargetNumber",
  "text": "Hello World!"
}
POST /api/sendMedia

إرسال وسائط عبر رقم مسجل (Form-Data).

  • senderPhone: الرقم المسجل الخاص بك النصي
  • number: رقم المستلم
  • mediaType: صورة/فيديو/الخ (حسب نوعك)
  • file: الملف المرفق
GET /api/status

تحميل حالة جميع الأرقام المسجلة في السيرفر.
